# 🛒 E-Commerce Full-Stack Platform

A modern, high-performance E-Commerce application built with the **MERN** stack. This project features a sleek, responsive interface, secure user management, and an intelligent AI chatbot to provide a premium shopping experience. 🚀

**🔗 Live Demo:** [https://ecomm-project-liard.vercel.app/](https://ecomm-project-liard.vercel.app/)

---

### ✨ Key Features

* **📱 Responsive Design:** Fully optimized for all devices using **Bootstrap 5**.
* **🔐 Secure Auth:** Robust **Sign-in** and **Sign-up** system with JWT-protected user sessions.
* **🤖 AI Chatbot:** Integrated smart assistant to handle customer queries and store navigation.
* **🛍️ Product Management:** Dynamic product listings, detailed views, and a fully functional shopping cart.
* **⚡ Real-time Updates:** Smooth data handling and state management for a seamless experience.

### 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1.  **Clone the Repo** 📥
    ```bash
    git clone [https://github.com/your-username/ecomm-project.git](https://github.com/your-username/ecomm-project.git)
    ```

2.  **Install Backend Dependencies** ⚙️
    ```bash
    cd backend
    npm install
    ```

3.  **Install Frontend Dependencies** 💻
    ```bash
    cd ../frontend
    npm install
    ```

4.  **Set Up Environment Variables** 🔑
    Create a `.env` file in the `/backend` folder:
    ```env
    MONGO_URI=your_mongodb_uri
    JWT_SECRET=your_secret_key
    PORT=5000
    ```

5.  **Launch the App** 🔥
    * Start Backend: `npm start`
    * Start Frontend: `npm start` (in a separate terminal)
